eve text editor
===============

eve is simple cross-platform text editor written in C++ with minimal dependencies. The executable is less than a hundred kilobytes. The editor works in a text console or terminal on six platforms - Windows, macOS, Linux, FreeBSD, Solaris, AIX.

eve stands for "excellent visual editor", "extremely versatile editor" or "eradicate vi and emacs" whichever you prefer :-)

Supported features:
* basic text editing operations
* multiple open documents
* open/save documents
* find/replace
* copy/delete/paste
* autoindent
* indent/unindent
* toggle comment
* go to line
* ability to execute make and use eve as a simple IDE
* basic Unicode support (can read/write UTF-8 and UTF-16, aware of multibyte characters)
* autocomplete based on words in open documents
